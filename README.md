# HotelPOS Web MVP

Système de gestion hôtelière avec point de vente intégré.

## Fonctionnalités
- Gestion de réception
- Check-in/Check-out
- Point de vente (POS)
- Facturation
- Administration

## Installation
```bash
pip install -r requirements.txt
python db_setup.py
python app.py
```

## Utilisation
Accédez à l'application via http://localhost:5000
